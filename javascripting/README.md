## Syntax

### Variables, Objects, Arrays, Strings, Numbers
```javascript
var pizza = {
  toppings: ['cheese', 'sauce', 'pepperoni'],
  crust: 'deep dish',
  serves: 2
};
```

### If Statement 
```javascript
if (n > 1) {
  console.log('the variable n is greater than 1.');
} else {
  console.log('the variable n is less than or equal to 1.');
}
```

### For Loop
```javascript
for (var i = 0; i < 10; i++) {
  // log the numbers 0 through 9
  console.log(i)
}
```

### Functions
```javascript
function example (x) {
  return x * 2;
}
```
